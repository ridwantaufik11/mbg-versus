# MBG Versus

Dengan dana Rp1,2 triliun per hari, MBG menjadi program pemerintah dengan anggaran terbesar dalam sejarah Indonesia. Tapi uang sebesar itu sebenarnya bisa untuk apa?

App ini memberi gambaran nyata: seberapa besar anggaran MBG jika dibandingkan dengan kebutuhan hidup rakyat Indonesia sehari-hari.


## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [Lucide React](https://lucide.dev/)

## Cara Jalankan Lokal

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Deploy ke Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push repo ini ke GitHub
2. Import di [vercel.com](https://vercel.com)
3. Klik Deploy — selesai

## Sumber Data

Data belanja BGN bersumber dari analisis **Project Multatuli** atas data LKPP/Inaproc TA 2025, dikonfirmasi Tempo, Kompas.id, dan Laporan Tahunan KPK 2025. Data pembanding dari BPS, Kemendikbud, dan sumber publik lainnya.
